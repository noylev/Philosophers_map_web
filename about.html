The Philosophers Map

search mapped information about more than a thousand philosophers! compare, reasearch and collect data easily

We are 3 students of "Computer Science" Bachlor degree in "Ben-Gurion University of the Negev". As part of the course "Digital Humanities For CS students" we created this project in order to help the philosophy department in reasearch and study. The main goal of our project is to visualize big amount of data about all the philosophers during all times.

The Process :
At first we imported data by asking queris with the tool of Wikidata. Then we analyzed the data in OpenRefine and fixed it. By using the OpenRefine tool we created a JSON file with all the data about each philosopher. We developed some scripts in order to get the right coordinates of birth place and death place. And finally we created functions that will help filter the data in the most efficient way.

In the main page we present the 'big picture' by showing each philosopher's marker on the global map, marked by his birth place. In addition to it, you can filter the philosopher's data by many criteria such as : Alive\Dead , Born Before, Male\Female etc.
